Descripcion Del Proyecto:


Hola a todos, en esta ocasión estaremos compartiendo con ustedes la presentación de nuestro proyecto final.
Mediante el siguiente proyecto queremos mostrar los grandes aprendizajes que gracias a nuestro gran maestro Juancito Pena fueron impartidos. Desde la creación de una base de datos desde cero hasta ser una gran base de datos para una empresa hasta reportes en ETL con C#, pudimos ver la importancia de la exportación e importación de una base de datos desde SQL a Excel y a Power BI el análisis de datos y la creación de reportes. Esperamos que este proyecto sirva como guía para nuevos lideres.

Impartida por el instructor Ing. Juancito Peña en la Universidad Dominicana O&M.
