Bienvenidos a la Presentacion Diseño de Centro de Datos

SQL Server

SQL (Structured Query Language) es un lenguaje de programación utilizado para gestionar y manipular datos en bases de datos relacionales.

Analisis con Excel

Un dashboard en Excel es una vista visual que resume información y datos importantes para la toma de decisiones..

Power BI

Un dashboard en Power BI es una vista interactiva y visual de los datos de una organización, diseñada para tomar decisiones informadas..

Analisis ETL

ETL (Extract, Transform, Load) es el proceso de extraer datos de múltiples fuentes, transformarlos a un formato uniforme y cargarlos en un destino para su análisis.
